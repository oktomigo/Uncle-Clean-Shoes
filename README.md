# 👟 Uncle Clean Shoes — Website Company Profile

Website company profile premium untuk **Uncle Clean Shoes**, layanan cuci sepatu, helm, dan tas profesional di Lamandau, Kalimantan Tengah.

## 🌐 Live Demo
> Setelah deploy ke GitHub Pages: `https://<username>.github.io/UNCLE_CLEANSHOES_LMD`

## 📁 Struktur Project
```
UNCLE_CLEANSHOES_LMD/
├── index.html          # Halaman utama
├── assets/
│   ├── css/
│   │   └── style.css   # Custom styling
│   ├── js/
│   │   └── main.js     # Interaksi & animasi
│   └── img/
│       ├── logo.webp
│       ├── 1_before.webp
│       ├── 1_after.webp
│       └── ...
└── .gitignore
```

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Bootstrap 5.3
- jQuery 3.7
- AOS (Animate On Scroll)
- SweetAlert2
- Font Awesome 6
- Google Fonts (Plus Jakarta Sans)

## 📱 Fitur
- Responsive mobile-first
- Sticky navbar dengan scroll effect
- Animasi AOS & counter
- Section before/after dengan foto asli
- FAQ accordion
- Booking WhatsApp dengan SweetAlert2
- Galeri Instagram grid
- Scroll to top button

## 🚀 Cara Deploy ke GitHub Pages
1. Push project ke repository GitHub
2. Buka **Settings** → **Pages**
3. Source: **Deploy from a branch** → branch `main` → folder `/ (root)`
4. Klik **Save** — website live dalam beberapa menit
